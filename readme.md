# Bus Schedule Scrapper


##### Information
- requirements: [requirements_01](requirements)
- folder hierarchy
  - `output`: output folder
  - `sample`: sample html of bus schedule
  - `sandbox`: just a junk snippets

##### Usage
```
$ python3 scrap_schedule.py --help
Usage: scrap_schedule.py [OPTIONS]

  Scaffold HTML table formed schedule into JSON

Options:
  --schedule_link TEXT  The link that leads directly to the schedule page
  --help                Show this message and exit.


$ python3 scrap_schedule.py --schedule_link http://www.pksgryfice.com.pl/uploads/images/rja/gryfice.html
Schedule link [http://www.pksgryfice.com.pl/uploads/images/rja/gryfice.html]: 
created: output/2019-05-18 15-00-52-524234.json

```
